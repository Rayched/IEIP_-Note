
## 데이터 정의어 (DDL)
- `Data Definition Language`의 약어
- 말 그대로 **"데이터를 정의하는 언어"**, 데이터를 담는 그릇을 정의하는 언어
- DB 생성하고, 데이터를 담을 Table을 정의하는 데 사용되는 명령어
- 특정 구조를 생성(Create), 삭제(Delete), 이름을 바꾸는 등(Update) <br/>
    데이터 구조와 관련된 명령어
- DDL의 대상에는 `Domain`, `Schema`, `Table`, `View`, `Index`가 있다.
